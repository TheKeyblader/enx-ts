# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2021-06-09

### Added

-   Added validation system.
-   Added mode to tree.
-   Added type helper schemaOf for zod.

### Fixed

-   Fixe duplicate attribute drawer.

## [0.3.0] - 2021-06-04

### Added

-   Support multiple same decorator on property.

### Changed

-   Updated draw chain to work with multiple same decorator.
-   Updated drawer to add drawer type with priority.
-   Change register drawer method.
-   decorators emit symbol as key.

### Removed

-   decorators no longer use symbol as key.

## [0.2.0] - 2021-05-28

### Added

-   Add hide decorators.
-   Add array support.

### Changed

-   Better property tree update.

## [0.1.0] - 2021-05-24

### Added

-   Initail release.
